# 🚀 Custom Space Invader  

Ce projet a été réalisé durant mes premières années d’apprentissage. Il s’agit d’une version personnalisée du célèbre jeu **Space Invaders**, adaptée à un univers de mon choix.

## 🎮 Fonctionnalités  
- Déplacement du joueur (gauche/droite)  
- Tir de projectiles  
- Vagues d’ennemis à éliminer  
- Score et difficulté progressive  

## 🛠️ Technologies utilisées  
- Javascript 

## 🚀 Lancer le jeu  
1. Cloner le dépôt :  
   ```bash
   git clone https://github.com/QuickoAbdul/Custom_Spaceinvader.git
   cd Custom_Spaceinvader
   ```
2. Lancer index.html en local
